Easy Conversion Of .Doc, .Pdf, .Txt,.Rtf.Htm Files In Desired Hindi Font

This feature allows the user to Convert his .doc,.pdf , .txt,.rtf   files in the desired font.

 

 

♠     Execute Agragami.exe

 

♠     Specify  the location of the Input file and the Output file and their respective format .

 

♠     Choose  the existing font type of the data and the desired output font.

 

♠     Press ‘Convert’ Button  and the data will be converted in the desired font.

 

♠     After conversion, a window specifying the current status, time elapsed, characters converted, errors and accuracy

     

       of the conversion will be displayed.

 

User Friendly Wizard

The feature allows the user to use a ‘step by step’  wizard to convert his/her file in the desired font.

 

 

♠    Execute Agragami.exe

 

♠    Press F1 Key to use a ‘step by step’  Wizard.

 

♠    Copy your text in Input.txt and close it.

 

♠    Select the desired Input and Output font.

 

♠    Press ‘Convert’ button to convert the data in the desired font.

 

♠    After conversion, a text file containing the data in the desired Output font will appear on the screen.

 

Enabling/Disabling ConvertDoc

Using ConvertDoc, Agragami Hindi Font Convertor can convert any .pdf,.doc,.txt,.rtf,.htm file in the desired font. Disabling it allows the user to convert only .txt files.

.

 

 

♠    Execute Agragami.exe

 

♠    Press F2 Key for ‘Settings’. Choose ‘YES’  if you want ot use ‘ Convert Doc ‘ ,else choose ‘NO’.

 

♠    Press ‘Enter’.
